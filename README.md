# PocketIQ 💰

> Know where every rupee goes.

PocketIQ is a modern, personal finance and expense tracking application designed to help users manage multiple bank accounts, monitor spending, set budgets,  and gain actionable financial insights through an intuitive and visually engaging experience.

Unlike traditional expense trackers, PocketIQ focuses on automation, intelligent analytics, and a Gen Z - friendly user experience with smooth animations, modern UI and smart budgeting tools

---

## 🎯 Vision

To build a production-ready fintech application that makes personal finance management simple, engaging and intelligent.

PocketIQ aims to combine the simplicity of UPI apps with the insight of modern budgeting platforms while providing users with complete control over their financial activities.

---

## ✨ Key Features

- 📱 Mobile OTP Authentication
- 🔐 Biometric Login (Fingerprint / Face ID)
- 🏦 Multiple Bank Account Management
- 💳 Credit Card Tracking
- 📊 Interactive Spending Analytics
- 🎯 Smart Budget Management
- 📩 Automatic Expense Detection (SMS + Notifications)
- 📄 Monthly Statement Generation
- 📈 AI-Powered Spending Insights
- 🌙 Beautiful Material 3 UI
- ⚡ Offline-First Experience
